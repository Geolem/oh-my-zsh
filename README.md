WHY?
====

为什么要自己建一个分支？  
主要是加了一些个人常用的设置（zshrc）。  
一旦换机器或重装系统什么的，不用再一个个想哪些常用的命令？该怎么设置了。  
只要下载了这个git库，把对应的文件复制过去就行了。

安装
====

关于 oh-my-zsh 的安装请参考原文档 [README.textile](./README.textile)

需要注意的地方：
====

0. 设置完成后需要重启

    因为一般默认的Shell都是bash，改变了用户默认的Shell，需要重新登陆。
